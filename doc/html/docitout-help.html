How This API Document Is Organized

This API (Application Programming Interface) document has pages corresponding to the items in the navigation bar, described as follows.

Overview

The Overview page is the front page of this API document and provides a list of all directories with a summary for each.

Directory

Each directory has a page that contains a list of its SAS files with a summary for each.

File

Each file has a page that contains the below information in the following order:-

  • File Information - General description of the file, author(s) and file creation date.
  • Include Summary - Summarizes all %include statements.
  • Libname Summary - Summarizes all libname statements.
  • Filename Summary - Summarizes all filename statements.
  • Macro Summary - Summarizes all %macro statements.
  • Macro Call Summary - List of all macro calls.
  • Include Detail - Detail information on all %include statements.
  • Libname Detail - Detail information on all libname statements.
  • Filename Detail - Detail information on all filename statements.
  • Macro Detail - Detail information on all %macro statements.
Each summary entry contains the first sentence from the detailed description for that item. All entries are sorted alphabetically. If a particular section (summary or detail) is empty, it will be not be shown in the page.
